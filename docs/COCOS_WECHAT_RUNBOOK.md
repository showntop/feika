# Cocos + 微信小游戏运行说明

## 开发运行

1. 用 Cocos Creator 3.8.x 打开仓库根目录。
2. 打开 `assets/scenes/Main.scene`。
3. 点击预览运行。

主场景只有一个 `Canvas`，挂载 `TenMinuteGame`。所有 UI 和玩法节点都由脚本生成，不需要手动拖组件。

## 微信小游戏构建

1. Cocos Creator 菜单进入「项目」->「构建发布」。
2. 平台选择「微信小游戏」。
3. 将 AppID 从 `touristappid` 改为你的小程序/小游戏 AppID。
4. 构建后，用微信开发者工具打开 Cocos 生成的微信小游戏构建目录。

## 当前 MVP 内容

- 竖屏 750x1334 UI。
- 7x7 合成盘。
- 商品、资金、人情三条合成链。
- 街边订单真实消耗盘面物品。
- 现金、体力、口碑、人脉资源。
- 第1晚短剧情推进。
- 纸箱摊升级小推车。
- 微信激励视频接口占位：`replace-with-your-ad-unit-id`。
