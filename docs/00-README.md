## 一句话定义

《重返2010：我的第一桶金》是一款面向微信小游戏用户的轻量合成经营游戏。玩家回到2010年，从夜市小摊和第一笔进货款开始，借助前世记忆抓住淘宝、智能机、微信、本地生活等时代红利，赚钱翻身、改善家庭、打脸小人，最终从小摊做到连锁生意。
