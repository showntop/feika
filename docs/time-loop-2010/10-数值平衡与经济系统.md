# 10. 数值平衡与经济系统

## 1. 核心经济模型

### 1.1 三货币体系

```yaml
现金 (Gold):
  用途: 店铺升级、剧情目标、生成器解锁
  获取: 订单完成、剧情事件、广告奖励
  稀缺性: 中等，主要进度货币

口碑 (Reputation):
  用途: 解锁高价值顾客、提升订单奖励
  获取: 完成高质量订单、剧情事件
  稀缺性: 高，决定解锁速度

人脉 (Connections):
  用途: 解锁特殊事件、贵人机会、批发渠道
  获取: 完成关键剧情、帮助NPC
  稀缺性: 很高，战略性资源
```

### 4.2 店铺等级效果

```yaml
等级效果计算公式:
  基础奖励 = 订单基础价值
  等级加成 = (店铺等级 - 1) * 20%
  最终奖励 = 基础奖励 * (1 + 等级加成)

示例:
  Lv1纸箱摊: 订单价值 * 100%
  Lv2小推车: 订单价值 * 120%
  Lv3夜市摊: 订单价值 * 140%
  Lv4临街小店: 订单价值 * 160%
  Lv5社区超市: 订单价值 * 180%
  Lv6连锁门店: 订单价值 * 200%
```

## 5. 体力与节奏控制

### 5.1 体力系统设计

```yaml
基础体力:
  上限: 60
  恢复速度: 1点/3分钟 (自然恢复)
  初始体力: 40

消耗规则:
  - 点击生成器: -1体力
  - 拖拽合成: 0体力
  - 完成订单: 0体力
  - 强制刷新订单: -5体力

恢复方式:
  - 自然恢复: 20点/小时
  - 广告恢复: 30体力/次 (每日5次)
  - 升级奖励: 补满体力
  - 章节完成: 补满体力
  - 每日登录: +20体力
```

### 5.2 游戏节奏分析

**单局游戏时长设计:**
```yaml
理想体验节奏:
  5分钟: 完成第一轮订单，理解核心玩法
  15分钟: 完成第一个剧情事件，获得成就感
  30分钟: 完成店铺第一次升级，看到成长
  60分钟: 完成第1章，获得完整体验

体力消耗节奏:
  - 前5分钟: 消耗15体力 (快速体验)
  - 5-15分钟: 消耗20体力 (深入探索)
  - 15-30分钟: 消耗25体力 (目标推进)
  - 30-60分钟: 消耗30体力 (完成章节)

广告介入节点:
  - 15分钟时: 第一次体力不足
  - 30分钟时: 关键材料缺一个
  - 45分钟时: 大订单想翻倍
```

## 6. 平衡性测试指标

### 6.1 关键平衡指标

```yaml
进度验证指标:
  - 新手教程完成率 > 85%
  - 第一次合成成功率 > 95%
  - 第一笔订单完成时间 < 2分钟
  - 第一次升级时间 < 15分钟
  - 第1章完成时间 < 60分钟

经济健康指标:
  - 现金缺口出现率 < 30%
  - 体力卡顿率 < 40%
  - 关键材料获得率 > 90%
  - 广告使用满意度 > 80%

数值平衡指标:
  - 合成成功率分布合理
  - 订单完成率 > 90%
  - 店铺升级时机适中
  - 剧情推进顺畅度 > 85%
```

### 6.2 调参方案

**如果进度过快:**
```yaml
调整方案:
  - 提升升级成本 10-15%
  - 降低生成器产量 10%
  - 增加高级合成难度
  - 提升体力消耗
```

**如果进度过慢:**
```yaml
调整方案:
  - 降低升级成本 10-15%
  - 提升生成器产量 10%
  - 增加广告奖励效果
  - 提升自然恢复速度
```
