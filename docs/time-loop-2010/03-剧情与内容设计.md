# 03. 剧情与内容设计

## 1. 剧情定位

本项目剧情不是传统长篇文字冒险，而是短剧式轻剧情：

- 冲突直接
- 目标明确
- 对话短
- 反转快
- 每章都有一个打脸或翻身节点

剧情的作用是给合成经营提供情绪动机，让玩家知道“为什么要赚这笔钱”。

## 2. 世界观

2026年，主角35岁，事业受挫，负债累累，家庭关系紧张。一次意外后，主角回到2010年老家。

此时家里正处在低谷：

- 父亲生病需要钱
- 家里欠着亲戚债
- 老房子可能被低价卖掉
- 亲戚和邻居都觉得主角没出息
- 曾经错过的时代机会即将出现

主角拥有前世记忆，知道2010年之后的多个普通人机会：

- 夜市和小商品
- 淘宝电商
- 智能手机配件
- 团购本地生活
- 微信朋友圈
- 快递和同城配送
- 直播电商前夜

## 3. 主角设定

### 3.1 默认主角

姓名：可由玩家自定义

年龄：

- 2026年：35岁
- 2010年：19岁或22岁

身份建议：

- 2010年：刚毕业/大学休学/回老家帮忙
- 2026年：普通打工人，懂一点互联网和生意，但不是精英人设

性格：

- 吃过亏，所以更稳
- 重视家人
- 有前世记忆，但仍要靠行动赚钱
- 能忍第一口气，但关键节点会反击

### 3.2 可选职业背景

后续可在开局提供轻量选择：

- 做过销售：顾客奖励提升
- 做过运营：口碑增长提升
- 做过程序/产品：电商和设备链效率提升
- 做过小生意：店铺升级成本降低

MVP阶段不建议做职业差异数值，避免复杂化。

## 4. 核心反派类型

反派要生活化，能让玩家快速理解。

### 4.1 势利亲戚

功能：

- 制造开局羞辱
- 形成“打脸”爽点
- 和债务、老房子、家庭压力绑定

示例台词：

“你家都这样了，还想着做生意？”

### 4.2 抢生意同行

功能：

- 制造经营竞争
- 推动玩家升级摊位和口碑

示例台词：

“这条街的生意，不是谁想做就能做的。”

### 4.3 黑心批发商

功能：

- 制造货源危机
- 引出贵人和人脉链

### 4.4 背刺朋友/合伙人

功能：

- 中期制造强冲突
- 推动玩家建立自己的品牌和团队

## 5. 短剧节奏模板

每章按以下节奏设计：

```text
1. 开场羞辱/危机
2. 给出明确目标
3. 玩家合成经营
4. 反派加压
5. 玩家用资源或前世记忆反转
6. 爽点结算
7. 新机会钩子
```

每个剧情节点对话控制在3-6句，避免长文本。

### 5.1 倒计时原则

章节可以在叙事上使用“三天内还债”“七天内拿下摊位”等压力，但不做真实墙钟倒计时和失败重开。

推荐做法：

- UI显示“第1天 / 第2天 / 第3天”的叙事进度。
- 天数随小关卡推进变化，不随真实时间流逝。
- 玩家下线、体力不足、没有看广告，都不会导致主线失败。
- 广告只提供“更快完成、更快打脸”的加速感，不提供“避免失败”的惩罚解除。

### 5.2 物品驱动原则

关键剧情不优先使用现金阈值触发，而是使用合成物提交触发。

原因：

- “提交第一桶金”比“现金达到5000”更有记忆点。
- 合成动作和剧情爽点能在同一刻发生。
- 物品链能让剧情目标更具体，减少纯刷钱感。

推荐格式：

```json
{
  "type": "item",
  "value": {
    "itemId": "money_5",
    "level": 5,
    "count": 1
  }
}
```

### 5.3 前世预知轻机制

“重生”不能只停留在台词里，首版用最轻机制落地：

- 每个小关卡最多出现1个“前世预知”提示。
- 预知不只是高亮，而是给玩家一个明确的信息特权。
- 信息特权包括：今日爆款、稀缺客户、价格预判。
- 完成后给额外现金、口碑或人脉奖励。

示例：

```text
前世记忆：今晚发卡会爆卖。
效果：本关一个“一袋小商品”相关订单奖励提高。
```

### 5.4 情绪平衡原则

首章可以用强冲突开局，但不能全程只有羞辱和打脸。

每章至少保留1-2个温暖时刻：

- 家人理解或支持
- 顾客感谢
- 老客户认可
- 贵人善意提醒

这些节点不削弱爽感，反而能让玩家觉得“值得把日子过好”，降低狗血疲劳。

## 6. MVP三章设计

### 第1章：三天赚出第一桶金

年份：2010

场景：老家小院、街边临时摊

核心目标：3天内赚出第一桶金，保住老房子。

开场：

- 主角从2026年醒来，发现自己回到2010年。
- 父亲住院，家里需要先交押金。
- 亲戚逼母亲签卖房协议，嘲笑主角没本事。

玩法目标：

- 合成商品链，卖小商品赚钱。
- 合成资金链，做出第一桶金。
- 完成第一笔收入后解锁人情链，拉到老客户并形成排队爽点。
- 完成剧情事件“第一桶金打脸”。

关键事件：

```text
事件1：先交住院押金
需要：攒钱罐 x1
奖励：剧情推进
```

```text
事件2：亲戚催债
需要：完成“先交住院押金”
奖励：建立三天叙事压力
```

```text
事件3：母亲的信任
需要：完成“亲戚催债”
奖励：现金 +50，口碑 +1
```

```text
事件4：街边试摊
需要：散货 x2
奖励：现金 +100
```

```text
事件5：前世预知：今晚会爆卖
需要：完成“街边试摊”
奖励：现金 +50，获得今日爆款提示
```

```text
事件6：第一笔收入
需要：小商品 x1
奖励：现金 +200，口碑 +2，解锁老式手机/人情链
```

```text
事件7：老客户带人排队
需要：老客户 x1 + 小商品 x1
奖励：现金 +200，口碑 +3，人脉 +1
```

```text
事件8：第一桶金打脸
需要：第一桶金 x1
奖励：完成第1章，解锁夜市摊
```

结尾爽点：

亲戚以为主角拿不出钱，主角提交“第一桶金”，当场还债并保住老房子。对方态度第一次变化。

章节钩子：

夜市管理员说：“今晚黄金摊位空出来了，但想要的人很多。”

### 第2章：夜市摆摊，赚第一桶金

年份：2010

场景：夜市

核心目标：拿下夜市黄金摊位，赚到第一桶金。

开场：

- 主角来到夜市摆摊。
- 老摊主不看好主角。
- 同行想用低价抢生意。

玩法目标：

- 解锁夜市摊车生成器。
- 新增人情链。
- 通过顾客和口碑提升订单价值。

关键事件：

```text
事件1：黄金摊位竞争
需要：进货款 x1 + 老客户 x1
奖励：获得黄金摊位
```

```text
事件2：同行压价
需要：热卖商品 x2 + 口碑 2
奖励：反击成功，订单奖励提升
```

```text
事件3：第一桶金
需要：第一桶金 x1
奖励：完成第2章，解锁门面线索
```

结尾爽点：

同行卖不动货，主角摊位排队。亲戚家的孩子偷偷来问能不能跟着干。

章节钩子：

一个老客户透露：商业街有家小门面急着转让。

### 第3章：盘下门面，开第一家店

年份：2011

场景：商业街小门面

核心目标：攒够转让费，开第一家店。

开场：

- 门面位置不错，但转让费高。
- 亲戚也看上这间店，想截胡。
- 主角知道这条街未来会变成商圈。

玩法目标：

- 新增设备链。
- 解锁小店货架、收银机。
- 完成门面转让事件。

关键事件：

```text
事件1：门面转让费
需要：第一桶金 x1 + 周转资金 x1
奖励：锁定门面
```

```text
事件2：亲戚截胡
需要：人脉 2 + 会员卡 x1
奖励：转让合同到手
```

```text
事件3：第一家店开业
需要：收银机 x1 + 热卖商品 x2
奖励：完成第3章，解锁淘宝机会
```

结尾爽点：

小店开业当天爆满，亲戚站在门口尴尬。主角收到一个消息：“现在网上也能开店，你听说过淘宝吗？”

## 7. 中长期章节方向

第4章：淘宝红利

- 手机壳、贴膜、充电器、维修服务

- 本地生活套餐、团购券、商家合作

第7章：微信朋友圈

- 私域客户、会员群、熟人订单

第8章：买第一套房

- 首付款、房产证、装修材料

- 品牌、合同、账本、证据链

- 店长、加盟合同、区域代理

## 8. 内容生产规范

### 8.1 对话长度

- 单次剧情对话：3-6句
- 单句不超过24字
- 避免长段解释时代背景

### 8.2 情绪密度

每章必须包含：

- 一个羞辱点
- 一个赚钱目标
- 一个困难升级
- 一个反转
- 一个钩子

### 8.3 合规原则

- 不使用真实企业名、真实企业家名。
- 不鼓励投机、诈骗、违法经营。
- 赚钱方式包装为勤劳、诚信、眼光、服务。
- 家庭困境避免过度卖惨。
