# 03. 剧情与内容设计

## 1. 剧情定位

本项目剧情不是传统长篇文字冒险，而是短剧式轻剧情：

- 冲突直接
- 目标明确
- 对话短
- 反转快
- 每章都有一个打脸或翻身节点

剧情的作用是给合成经营提供情绪动机，让玩家知道“为什么要赚这笔钱”。

## 2. 世界观

2026年，主角35岁，事业受挫，负债累累，家庭关系紧张。一次意外后，主角回到2010年老家。

此时家里正处在低谷：

- 父亲生病需要钱
- 家里欠着亲戚债
- 老房子可能被低价卖掉
- 亲戚和邻居都觉得主角没出息
- 曾经错过的时代机会即将出现

主角拥有前世记忆，知道2010年之后的多个普通人机会：

- 夜市和小商品
- 淘宝电商
- 智能手机配件
- 团购本地生活
- 微信朋友圈
- 快递和同城配送
- 直播电商前夜

## 3. 主角设定

### 3.1 默认主角

姓名：可由玩家自定义

年龄：

- 2026年：35岁
- 2010年：19岁或22岁

身份建议：

- 2010年：刚毕业/大学休学/回老家帮忙
- 2026年：普通打工人，懂一点互联网和生意，但不是精英人设

性格：

- 吃过亏，所以更稳
- 重视家人
- 有前世记忆，但仍要靠行动赚钱
- 能忍第一口气，但关键节点会反击

### 3.2 可选职业背景

后续可在开局提供轻量选择：

- 做过销售：顾客奖励提升
- 做过运营：口碑增长提升
- 做过程序/产品：电商和设备链效率提升
- 做过小生意：店铺升级成本降低

MVP阶段不建议做职业差异数值，避免复杂化。

## 4. 核心反派类型

反派要生活化，能让玩家快速理解。

### 4.1 势利亲戚

功能：

- 制造开局羞辱
- 形成“打脸”爽点
- 和债务、老房子、家庭压力绑定

示例台词：

“你家都这样了，还想着做生意？”

### 4.2 抢生意同行

功能：

- 制造经营竞争
- 推动玩家升级摊位和口碑

示例台词：

“这条街的生意，不是谁想做就能做的。”

### 4.3 黑心批发商

功能：

- 制造货源危机
- 引出贵人和人脉链

### 4.4 背刺朋友/合伙人

功能：

- 中期制造强冲突
- 推动玩家建立自己的品牌和团队

## 5. 短剧节奏模板

每章按以下节奏设计：

```text
1. 开场羞辱/危机
2. 给出明确目标
3. 玩家合成经营
4. 反派加压
5. 玩家用资源或前世记忆反转
6. 爽点结算
7. 新机会钩子
```

每个剧情节点对话控制在3-6句，避免长文本。

## 6. MVP三章设计

### 第1章：重回2010，家里欠债

年份：2010

场景：老家小院、街边临时摊

核心目标：3天内凑齐5000元，避免家里卖老房子。

开场：

- 主角从2026年醒来，发现自己回到2010年。
- 父亲病倒，家里需要一笔钱。
- 亲戚上门催债，嘲笑主角没本事。

玩法目标：

- 合成商品链，卖小商品赚钱。
- 合成资金链，凑出周转资金。
- 完成剧情事件“还上第一笔债”。

关键事件：

```text
事件1：亲戚催债
需要：攒钱罐 x1
奖励：剧情推进
```

```text
事件2：街边试摊
需要：小商品 x2
奖励：现金 +100，解锁旧纸箱升级
```

```text
事件3：第一笔还款
需要：周转资金 x1
奖励：完成第1章，解锁夜市摊
```

结尾爽点：

亲戚以为主角拿不出钱，主角当场还上第一笔债。对方态度第一次变化。

章节钩子：

夜市管理员说：“今晚黄金摊位空出来了，但想要的人很多。”

### 第2章：夜市摆摊，赚第一桶金

年份：2010

场景：夜市

核心目标：拿下夜市黄金摊位，赚到第一桶金。

开场：

- 主角来到夜市摆摊。
- 老摊主不看好主角。
- 同行想用低价抢生意。

玩法目标：

- 解锁夜市摊车生成器。
- 新增人情链。
- 通过顾客和口碑提升订单价值。

关键事件：

```text
事件1：黄金摊位竞争
需要：进货款 x1 + 老客户 x1
奖励：获得黄金摊位
```

```text
事件2：同行压价
需要：热卖商品 x2 + 口碑 2
奖励：反击成功，订单奖励提升
```

```text
事件3：第一桶金
需要：第一桶金 x1
奖励：完成第2章，解锁门面线索
```

结尾爽点：

同行卖不动货，主角摊位排队。亲戚家的孩子偷偷来问能不能跟着干。

章节钩子：

一个老客户透露：商业街有家小门面急着转让。

### 第3章：盘下门面，开第一家店

年份：2011

场景：商业街小门面

核心目标：攒够转让费，开第一家店。

开场：

- 门面位置不错，但转让费高。
- 亲戚也看上这间店，想截胡。
- 主角知道这条街未来会变成商圈。

玩法目标：

- 新增设备链。
- 解锁小店货架、收银机。
- 完成门面转让事件。

关键事件：

```text
事件1：门面转让费
需要：第一桶金 x1 + 周转资金 x1
奖励：锁定门面
```

```text
事件2：亲戚截胡
需要：人脉 2 + 会员卡 x1
奖励：转让合同到手
```

```text
事件3：第一家店开业
需要：收银机 x1 + 热卖商品 x2
奖励：完成第3章，解锁淘宝机会
```

结尾爽点：

小店开业当天爆满，亲戚站在门口尴尬。主角收到一个消息：“现在网上也能开店，你听说过淘宝吗？”

## 7. 中长期章节方向

第8章：买第一套房

第9章：合伙人背刺

- 品牌、合同、账本、证据链

第10章：城市连锁

- 店长、加盟合同、区域代理

## 8. 内容生产规范

### 8.1 对话长度

- 单次剧情对话：3-6句
- 单句不超过24字
- 避免长段解释时代背景

### 8.2 情绪密度

每章必须包含：

- 一个羞辱点
- 一个赚钱目标
- 一个困难升级
- 一个反转
- 一个钩子

### 8.3 合规原则

- 不使用真实企业名、真实企业家名。
- 不鼓励投机、诈骗、违法经营。
- 赚钱方式包装为勤劳、诚信、眼光、服务。
- 家庭困境避免过度卖惨。
