# 快速原型使用指南

## 🎮 快速原型概述

这是一个最小化可玩版本，让你能快速体验核心游戏玩法。

### 包含功能：
- ✅ 基础游戏界面（标题、金币、分数显示）
- ✅ 2x2合成网格（快速原型规模）
- ✅ 简单的订单系统（随机订单生成）
- ✅ 点击交互和反馈动画
- ✅ 完成订单获得奖励机制

## 🚀 如何在Cocos Creator中运行

### 1. 刷新Cocos Creator项目
在Cocos Creator编辑器中：
- 按 `Cmd+R` 或点击"刷新"按钮
- 新场景和脚本会自动识别

### 2. 打开主游戏场景
在资源管理器中：
- 找到 `assets/scenes/MainGameScene.scene`
- 双击打开场景

### 3. 配置场景组件（首次需要）
MainGameScene场景需要手动配置UI组件绑定：

#### 步骤：
1. 在层级管理器中选择Canvas节点
2. 在检查器中添加组件：
   - 搜索 "MainGameSceneController"
   - 添加组件到Canvas

3. 创建UI节点结构：
```
Canvas/
├── TitleLabel (Label组件)
├── CoinsLabel (Label组件)
├── ScoreLabel (Label组件)
├── OrderDescriptionLabel (Label组件)
├── MergeGridContainer (空节点)
└── CompleteOrderButton (Button组件)
```

4. 绑定组件引用：
- 在MainGameSceneController组件中
- 将对应的UI节点拖拽到属性框中

### 4. 运行预览
- 点击工具栏的"运行预览"按钮
- 浏览器会打开并显示游戏界面

## 🎯 游戏玩法说明

### 基础玩法：
1. **观察订单**：屏幕显示当前订单需求
2. **点击网格**：点击2x2网格中的物品
3. **完成订单**：点击"完成订单"按钮获得奖励
4. **循环游戏**：完成后自动生成新订单

### 游戏元素：
- 🍎 苹果：基础物品
- 🍑 桃子：基础物品
- 🍇 葡萄：基础物品
- 💰 金币：完成订单获得
- ⭐ 分数：每次完成订单增加

## 📊 快速原型特性

### 简化的游戏循环：
```
开始游戏 → 生成订单 → 点击物品 → 完成订单 → 获得奖励 → 新订单
```

### 核心系统集成：
- ✅ GameManager（游戏管理器）

- ✅ MergeSystem（合成系统框架）

## 🛠️ 技术架构说明

### 文件结构：
```
assets/
├── scenes/
│   ├── LaunchScene.scene          # 启动场景
│   └── MainGameScene.scene        # 主游戏场景（新增）
├── scripts/
│   ├── core/                      # 核心系统
│   │   ├── GameManager.ts
│   │   ├── EventManager.ts
│   │   └── ...
│   └── gameplay/                  # 游戏玩法
│       ├── MainGameSceneController.ts  # 主游戏控制器（新增）
│       ├── merge/
│       │   └── MergeSystem.ts
│       └── business/
│           └── BusinessSystem.ts
```

### 关键设计：
1. **场景-控制器分离**：每个场景有独立控制器
2. **系统解耦**：核心系统与UI分离
3. **事件驱动**：使用EventManager进行通信
4. **可扩展性**：预留了完整的扩展接口
