# Cocos Creator 项目快速启动指南

## 🚀 立即开始（3步验证）

### 第1步：在Cocos Creator中打开项目
1. 启动 **Cocos Creator 3.x**（推荐3.8或更高版本）
2. 选择 "打开其他项目"
3. 选择项目目录：`/Users/denny/Work/feica`
4. **重要**：如果出现版本选择，请选择 **Cocos Creator 3.x**

### 第2步：验证项目结构
在Cocos Creator编辑器中检查：
- ✅ **资源管理器** 中能看到 `assets/scenes/LaunchScene.scene`
- ✅ **层级管理器** 中显示场景包含 Canvas 和 Camera 节点
- ✅ **控制台** 无红色错误信息（可能有Cocos相关的警告，可忽略）

### 第3步：运行项目预览
1. 点击编辑器顶部的 **"运行预览"** 按钮
2. 浏览器会自动打开并显示启动场景
3. 检查控制台是否有以下日志：
   ```
   [LaunchScene] 启动场景加载
   [GameApp] onLoad - Cocos Creator组件加载
   [LaunchScene] 开始启动流程
   ```

## ⚠️ 常见问题

### Q1: "找不到模块 'cc'" 错误
**原因**: 这是预期行为，不是真正错误
**解决方案**: 这些文件需要在Cocos Creator编辑器中编译，编辑器会自动处理

### Q2: 项目无法打开，显示配置错误
**原因**: 可能使用了错误的Cocos Creator版本
**解决方案**: 确保使用 Cocos Creator 3.x，而不是 2.x

### Q3: 运行时黑屏
**原因**: 当前只有启动场景，还没有UI内容
**状态**: 正常现象，下一步会创建UI
