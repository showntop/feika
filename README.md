# 🎮 重返2010：我的第一桶金

一款"重生短剧爽文 + 二合玩法 + 轻经营成长"的微信小游戏。玩家回到2010年，从被亲戚看不起、家里欠债、人生低谷开始，通过合成商品、经营摊位、抓住时代机会，一步步赚钱翻身、买房开店、改善家人命运。

## 🌟 项目特色

- **创新题材组合**: 短剧情感 + 合成玩法 + 2010年代怀旧
- **三货币经济系统**: 现金、口碑、人脉平衡设计
- **时代红利体验**: 夜市、淘宝、智能机、微信等真实商业机会
- **情感驱动剧情**: 从负债到翻身的爽文式体验
- **轻量化操作**: 7x7合成盘，简单易上手

## 📋 项目概述

### 核心玩法
- **合成系统**: 二合操作，从散货到招牌商品的成长链
- **经营系统**: 顾客订单、店铺升级、口碑管理
- **剧情系统**: 短剧式事件，章节制故事推进
- **时代机遇**: 2010-2020真实商业机会再现

### 技术栈
- **游戏引擎**: Cocos Creator 3.x
- **开发语言**: TypeScript 5.x
- **目标平台**: 微信小游戏
- **架构模式**: MVC + 事件驱动

## 🎯 核心系统

### 1. 合成系统 (MergeSystem)
- 7x7合成盘管理
- 物品拖拽合成逻辑
- 生成器系统
- 体力管理

### 2. 经营系统 (BusinessSystem)
- 订单生成和管理
- 店铺升级系统
- 资源管理（现金、口碑、人脉）
- 顾客类型解锁

### 3. 剧情系统 (StorySystem)
- 章节管理
- 剧情事件触发
- 对话系统
- 目标完成检测

### 4. 游戏管理器 (GameManager)
- 系统协调
- 游戏状态管理
- 玩家数据持久化
- 游戏主循环

## 📊 游戏数值

### MVP范围
- **章节**: 3章（重回2010、夜市摆摊、盘下门面）
- **合成链**: 5级商品、资金、人情链
- **店铺等级**: 6级（纸箱摊 → 连锁门店）
- **订单类型**: 4种顾客类型
- **剧情事件**: 第1章包含6个关键事件

### 经济模型
- **初始现金**: 100元
- **初始体力**: 40点（上限60点）
- **体力恢复**: 20点/小时
- **第1章目标**: 3天内赚5000元

## 🎨 美术风格

- **视觉风格**: 下沉友好、接地气、2010年代怀旧
- **UI原则**: 简洁直观、图标大、目标明确
- **场景设计**: 从纸箱摊到连锁门店的可视化成长
- **年代元素**: 老式手机、夜市灯泡、纸质单据等

## 💰 商业化

### IAA广告变现
- **体力恢复**: 看广告获得30体力
- **材料补齐**: 缺1个材料时看广告补齐
- **奖励翻倍**: 大订单完成后看广告翻倍
- **每日宝箱**: 每日签到看广告开启

### 数据指标
- **目标次日留存**: >25%
- **目标7日留存**: >8%
- **人均广告次数**: 3-5次/日
- **广告完成率**: >75%
